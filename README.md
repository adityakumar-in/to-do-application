# To-do-application

A simple and responsive Todo Web Application built using PHP, MySQL, HTML, and CSS. The application allows users to create an account, log in, and manage their personal tasks easily.

## Features

* User authentication system with registration and login functionality
* Separate task storage for every user
* Add new tasks
* Mark tasks as completed
* Delete tasks
* Prevent duplicate task entries
* Task creation date and time display
* Responsive design for desktop and mobile devices
* Clean and user-friendly interface

## Technologies Used

Frontend:

* HTML
* CSS

Backend:

* PHP

Database:

* MySQL / MariaDB

## About This Project

This project was developed as a practice full-stack web application to understand how frontend, backend, and databases work together.

It includes important web development concepts such as user authentication, session management, CRUD operations, database handling, and responsive web design.

The goal of this project was to build a functional application with a simple interface while improving practical development skills.
